# Vacuum Resistance Thermobaric Patch

A lightweight compatibility patch for RimWorld 1.6.

## Description

This mod makes More Injuries (Continued) respect the game's Vacuum Resistance stat.

Pawns wearing power armor or vacsuits are less likely to suffer lung collapse from bomb and Combat Extended thermobaric explosions.

Other blast injuries are unaffected.

## Requirements

- Harmony
- Odyssey
- More Injuries 
- Combat Extended (optional)

## Installation

Extract the mod into your RimWorld `Mods` folder and enable it after More Injuries (Continued).

## Compatibility

- RimWorld 1.6
- Safe to add or remove from existing saves.

## Source

This repository contains both the mod's source code and the compiled assembly.
